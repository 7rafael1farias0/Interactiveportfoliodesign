# 💻 Meu Portfólio Interativo

Bem-vindo ao repositório do meu portfólio! Este projeto foi desenvolvido para apresentar minhas habilidades, projetos e experiências na área de desenvolvimento de software de forma interativa e moderna.

O design original foi criado no Figma e pode ser visualizado [neste link](https://www.figma.com/design/Xo0oEgUqakRveHsb2xywLh/Interactive-Portfolio-Design).

🚀 **[Acesse o Portfólio Online Aqui](https://7rafael1farias0.github.io/Interactiveportfoliodesign/)**

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes ferramentas:

* **[React](https://reactjs.org/)** - Biblioteca JavaScript para construção da interface.
* **[Vite](https://vitejs.dev/)** - Ferramenta de build super rápida.
* **[TypeScript](https://www.typescriptlang.org/)** - Superset de JavaScript que adiciona tipagem estática.
* **[Tailwind CSS](https://tailwindcss.com/)** - Framework CSS utilitário para estilização rápida e responsiva.
